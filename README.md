# Touch Grass!

Hover your mouse over the grass, go outside for a bit, and a garden will grow by the time your back!
Alternatively, its a nice background; you can adjust the garden and weather in the settings, or input your zip code to retrieve the local weather.

## Notes & Attributions

We use an audio library (Howler.js)---if the website breaks, run npm install.

We use the [Open-Meto API](https://open-meteo.com/) to retrieve weather data.

We also pasted a mulberry32 PRNG function. There's plenty of details if you search for it, and you can find the JS implementation [here](https://github.com/cprosche/mulberry32)

## AI Use

This is all of our first webdev projects, so AI was really useful for getting started, and learning about tools it would've been difficult to find otherwise!
